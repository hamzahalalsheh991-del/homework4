# Проектирование документной модели для фитнес-трекера

## Выбор коллекций
Система состоит из трёх основных сущностей:
- **Пользователь** – хранит персональные данные и учётные записи.
- **Упражнение** – справочник упражнений (название, описание, группа мышц).
- **Тренировка** – запись о проведённой тренировке, связывающая пользователя с набором выполненных упражнений.

Каждая сущность размещена в отдельной коллекции: `users`, `exercises`, `workouts`.

## Структура документов

### `users`
| Поле       | Тип     | Описание                       |
|------------|---------|--------------------------------|
| _id        | ObjectId| Уникальный идентификатор       |
| login      | string  | Логин (уникальный)             |
| firstName  | string  | Имя                            |
| lastName   | string  | Фамилия                        |
| email      | string  | Адрес электронной почты        |
| age        | number  | Возраст (18–100)               |
| createdAt  | date    | Дата регистрации               |
| updatedAt  | date    | Дата последнего обновления     |

### `exercises`
| Поле          | Тип     | Описание                          |
|---------------|---------|-----------------------------------|
| _id           | ObjectId| Уникальный идентификатор          |
| name          | string  | Название упражнения (приседания)  |
| description   | string  | Техника выполнения                |
| muscleGroup   | string  | Группа мышц (грудь, спина, ноги)  |
| difficulty    | string  | Уровень сложности (beginner, pro) |
| createdBy     | ObjectId| Ссылка на пользователя (кто создал)|
| createdAt     | date    | Дата создания                     |

### `workouts`
| Поле      | Тип          | Описание                                        |
|-----------|--------------|-------------------------------------------------|
| _id       | ObjectId     | Уникальный идентификатор                        |
| userId    | ObjectId     | Ссылка на пользователя (ref)                    |
| date      | date         | Дата тренировки                                 |
| duration  | number       | Длительность в минутах                          |
| notes     | string       | Заметки (самочувствие, погода и т.п.)           |
| exercises | Array[Object]| Встроенный список выполненных упражнений        |
| createdAt | date         | Дата создания записи                            |
| updatedAt | date         | Дата обновления                                 |

**Элемент массива `exercises`:**
```js
{
  exerciseId: ObjectId,   // ссылка на упражнение из коллекции exercises
  sets: number,           // количество подходов
  reps: number,           // количество повторений
  weight: number,         // вес (кг)
  restTimeSec: number     // отдых между подходами (сек)
}
